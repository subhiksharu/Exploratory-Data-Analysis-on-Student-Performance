**Student Performance Analysis Using Exploratory Data Analysis**

This project performs Exploratory Data Analysis on student performance data to understand the factors 
that influence academic results. The dataset contains student-related features such as study hours, attendance percentage, 
previous marks, final score, parental education, internet access, extra classes, sleep hours, age, and gender.

The analysis focuses on identifying patterns, relationships, and trends in student academic performance. 
It includes data cleaning, missing value checking, statistical summaries, categorical analysis, correlation analysis, 
data visualization, and interpretation of key influencing factors.

**Tools Used**
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

**Key Objective**

To analyze student performance data and identify the major academic, personal, and family-related factors that affect final exam scores.

**Project Workflow**
- Import required Python libraries
- Load the student performance dataset
- Understand dataset structure using `head()`, `shape`, `info()`, and `describe()`
- Check for missing values and duplicate records
- Analyze numerical features such as study hours, attendance, previous marks, final score, and sleep hours
- Analyze categorical features such as gender, parental education, internet access, and extra classes
- Create visualizations to identify patterns and trends
- Use correlation analysis to find relationships between numerical variables
- Compare average scores across different student groups
- Summarize key findings and conclusions

**Key Features Analyzed**
- Study hours per day
- Attendance percentage
- Previous marks
- Final exam score
- Parental education
- Internet access
- Extra classes
- Sleep hours
- Age
- Gender

**Visualizations Used**
- Histogram to show distribution of final scores
- Scatter plot for study hours vs final score
- Scatter plot for attendance vs final score
- Bar chart for parental education vs average final score
- Box plot for extra classes vs final score
- Correlation heatmap for numerical features
- Count plot for categorical variables

**Key Insights**
- Study hours, attendance, and previous marks strongly affect final scores.
- Students who study more and attend regularly usually perform better.
- Extra classes and internet access support better academic performance.
- Parental education may provide better learning support at home.
- Balanced sleep habits can positively influence student results.
- Low attendance and fewer study hours are linked with lower scores.
- Students with higher previous marks are more likely to score well in final exams.
- Academic performance depends on both personal habits and learning resources.
- Regular practice and consistent study routines can improve student outcomes.

**Expected Outcome**

This project helps develop analytical thinking and data exploration skills. It also improves 
understanding of how different factors such as study habits, attendance, parental education, 
and learning resources contribute to student academic success.

**Conclusion**

The analysis shows that student performance is mainly influenced by study hours, attendance, 
previous marks, extra classes, parental education, internet access, and sleep habits. Among these factors, 
study hours, attendance, and previous marks appear to have the strongest effect on final exam scores. 
Therefore, students can improve their academic performance by maintaining regular attendance, increasing study time, 
using available learning resources, and following healthy sleep habits.
